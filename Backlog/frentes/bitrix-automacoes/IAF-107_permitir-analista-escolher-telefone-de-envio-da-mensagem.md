# IAF-107 — [Bitrix Cobrança 4.0] Permitir analista escolher telefone de envio da mensagem

## Descrição (espelho — fonte: jira)

> Origem: Jira IAF-107 · categoria: null

Disponibilizar no card uma lista com todos os telefones válidos do contrato para que o **analista escolha** qual número utilizar no envio da mensagem de acionamento.

Justificativa: se o cliente tiver bloqueado o número oficial de cobrança, o analista pode optar por um telefone corporativo alternativo, aumentando a probabilidade de sucesso do contato.

Depende do card \[Bitrix Cobrança 4.0\] Exibir todos os telefones do módulo financeiro no card.
