# Frente — Estratégica (Transversal)

Itens transversais que não se encaixam em uma frente única — processo, narrativa institucional, métricas de jornada, dependências cruzadas entre frentes.

**Sponsor padrão:** Diretoria Finza / João Vinícius
**ID prefix:** `BST##`

## Escopo

- Coleta de NPS pós-jornada Finza (via IA)
- Narrativa institucional do impacto IA no negócio
- Definição de processo (raias, fluxos, governança do squad)
- Decisões transversais que afetam múltiplas frentes

## Critério de inclusão

Um item entra aqui SE:
- Não tem uma frente operacional única (toca 3+ frentes), OU
- É decisão de governança/processo, OU
- É comunicação executiva (deck, narrativa para diretoria), OU
- É métrica de jornada completa (NPS, CSAT, satisfação)

## Fora de escopo

- Qualquer item que tenha frente operacional clara (vai para a frente específica)
