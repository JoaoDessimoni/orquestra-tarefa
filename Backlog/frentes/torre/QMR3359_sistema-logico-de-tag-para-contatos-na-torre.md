# QMR3359 — Sistema lógico de Tag para contatos na Torre

## Descrição (espelho — fonte: quimera+jira)

> Origem: Quimera #3359 · Jira IAF-110 · categoria: TI

### Resumo

Implementar um campo de tag nos contatos da Torre. Isso permitirá melhorar os filtros e criar a lógica "not" para os filtros.

### Contexto

A proposta visa aprimorar a organização dos contatos através de tags. Com isso, será possível realizar filtragens mais eficazes.

### Critérios de aceitação

* Implementar um campo de tag nos contatos.
* Melhorar os filtros existentes com base nas tags.
* Criar a lógica "not" para os filtros.
* Definir a forma de adicionar as tags.
